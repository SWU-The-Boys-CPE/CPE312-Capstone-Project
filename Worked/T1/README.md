## üìñ Key Document: Project Definition

The main document `10Traffic-Transport-Capstone.md` includes:

### 1. Problem Definition
- Bangkok's congestion costs ~97 million THB daily
- Average congestion index: 38.88, Peak: 162.13
- Core question: Can we predict daily TCI using historical data and weather?

### 2. Research Questions
1. What are the daily, weekly, and seasonal patterns in Bangkok's TCI?
2. How do weather conditions correlate with congestion levels?
3. Which ML model provides the best prediction accuracy?
4. Which features are most predictive of congestion?

### 3. Objectives
1. Collect and clean traffic + weather data
2. Perform exploratory data analysis
3. Engineer predictive features
4. Train and compare ML models
5. Evaluate model performance

### 4. Data Sources
| Dataset | Records | Description |
|---------|---------|-------------|
| Bangkok Traffic Congestion Index | 1,682 | Daily TCI values (2019-2025) |
| Bangkok Weather Data | 1,682 | Temperature, humidity, precipitation |

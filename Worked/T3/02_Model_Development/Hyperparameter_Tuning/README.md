# Hyperparameter Tuning

This directory contains hyperparameter tuning results and configurations.

## Files

| File | Purpose |
|------|---------|
| `tuning_results.csv` | All tuning experiments |
| `best_params.yaml` | Best parameters per model |

## Tuning Methods Used

1. **Grid Search** - ARIMA, Random Forest
2. **Random Search** - XGBoost (initial)
3. **Bayesian Optimization** - LSTM, XGBoost (fine-tuning)
